BookWright

BookWright è l'unico strumento di desktop publishing creato per la pubblicazione di libri, riviste su carta e e-book con aspetto professionale. Non dovrai consultare un designer, conoscere codici né acquistare software costosi.

Scarica BookWright


Uno strumento di creazione per la creatività che c’è in ognuno di noi

BookWright è un'applicazione potente per design e layout multifunzione per la creazione di fotolibri, libri tascabili, riviste e e-book. È semplice da usare e gratuito. Ecco le funzionalità:

  • Layout gratuiti, scaricabili e progettati con aspetto professionale (o possibilità di creare i tuoi)
  • Stampa di libri, e-book o PDF con lo stesso file di progetto
  • ISBN univoco e gratuito per ogni versione del libro (per un valore di $ 125).
  • Stampa super fedele grazie al profilo colori incorporato per le nostre stampanti
  • Riquadri di testo con adattamento automatico
  • Importazione Rich Text Format (RTF) per libri basati sul testo
  • Importazione dei file Blurb esistenti (creati con BookSmart o Bookify)

BookWright è probabilmente molto intuitivo, ma abbiamo comunque creato una guida dettagliata per aiutarti a iniziare.
