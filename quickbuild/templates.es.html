Plantillas para libros y revistas diseñadas por profesionales

No hace falta que sea diseñador para crear una publicación de aspecto profesional; nosotros ya hemos hecho el trabajo pesado por usted con nuestras plantillas de libros de fotografías y revistas. Cada archivo de descarga gratuita incluye 20 páginas (o más) para que pueda comenzar. Puede usar los diseños tal como están o personalizarlos.

Nuestras plantillas de composición se crearon con BookWright y nuestro plug-in para Adobe InDesign en mente. Hay plantillas disponibles para libros de fotografías, revistas, libros de cocina y catálogos en formato vertical o apaisado. Para cada tipo de libro, puede elegir entre tres diseños de plantilla únicos.

  • Plantillas para InDesign
  • Plantillas de BookWright

Vertical

Las plantillas de libros verticales contienen una variedad de composiciones que combinan fotografías y texto: una opción ideal para libros con gran riqueza gráfica. Presente sus fotografías, ilustraciones y gráficos (y compleméntelos con texto) con estas ingeniosas composiciones. Seleccione uno de los estilos a continuación para comenzar con las páginas que necesita para crear un hermoso libro vertical.

Esta plantilla contiene todas las composiciones necesarias para crear un libro apaisado con gran riqueza gráfica. Elija entre una variedad de composiciones que contienen imágenes y texto. Este estilo es una escala de grises básica que le permite definir y aplicar su propia paleta de colores.

Formatos compatibles:
sobrecubierta de tapa dura, tapa dura impresa, tapa blanda y ebook
Recuento de plantillas:
20 páginas y cubiertas únicas
Fuentes utilizadas:
Playfair, Garamond
Paleta:
escala de grises
Usos sugeridos:
recetas familiares, recetas estacionales, recetas de restaurante
Versión admitida:
Blurb BookWright 1.0 o superior
Software:
Blurb BookWright

Descargar plantilla

  1. ¿Está listo para utilizar esta plantilla? Haga clic en el botón Descargar plantilla.
  2. Localice el archivo .blurb en la carpeta Descargas o en el Escritorio.
  3. Haga doble clic en el archivo .blurb para ejecutar BookWright.
  4. Seleccione Guardar como e introduzca un nombre de archivo original para su libro.
  5. Revise los consejos sobre plantillas para principiantes para comenzar.
