Plantillas para libros y revistas diseñadas por profesionales

Las plantillas gratuitas y descargables para revistas y libros de fotografías de Blurb son perfectas para diseñadores y aficionados. Puede usar las plantillas tal como están o personalizarlas para su proyecto.

Nuestras plantillas de composición para principiantes son compatibles con Blurb BookWright y Adobe InDesign CC (con el plug-in de Blurb para InDesign instalado). Los usuarios de Adobe Creative Suite versiones 3 a 6 deben usar las plantillas en blanco disponibles después de instalar el plug-in.

  • Plantillas para BookWright
  • Plantillas para InDesign CC

Vertical

Las plantillas de libros verticales contienen una variedad de composiciones que combinan fotografías y texto: una opción ideal para libros con gran riqueza gráfica. Presente sus fotografías, ilustraciones y gráficos (y compleméntelos con texto) con estas ingeniosas composiciones. Seleccione uno de los estilos a continuación para comenzar con las páginas que necesita para crear un hermoso libro vertical.

Descargar plantilla

Nuestras plantillas de composición para principiantes requieren Adobe InDesign CC (con el plug-in de Blurb para InDesign instalado). Los usuarios de Adobe Creative Suite versiones 3 a 6 deben usar las plantillas en blanco disponibles después de instalar el plug-in.

  1. Instale el plug-in de InDesign.
  2. ¿Está listo para utilizar esta plantilla? Haga clic en el botón Descargar plantilla.
  3. Localice el archivo .zip para descomprimir los archivos de páginas y cubierta de la plantilla.
  4. Abra esos archivos en InDesign CC para comenzar a crear su libro.
  5. Revise los consejos sobre plantillas para principiantes para comenzar.
