BookWright

BookWright es el único software de autoedición diseñado exclusivamente para crear libros, revistas e ebooks de aspecto profesional. No necesita contratar a un diseñador, conocer una línea de código ni comprar software costoso.

Una herramienta de creación para el creativo que llevamos dentro

BookWright es una potente aplicación de diseño y composición con múltiples características para crear libros de fotografías, libros comerciales, revistas e ebooks. Es fácil de utilizar —y es gratuita—. Las características incluyen lo siguiente:

  • Composiciones gratuitas y descargables diseñadas profesionalmente (y la capacidad de crear las suyas con facilidad)
  • Libro impreso, ebook y PDF a partir del mismo archivo de proyecto
  • Un ISBN único y gratuito para cada versión de su libro (por un valor de $125)
  • Material impreso tal como se ve en pantalla gracias a un perfil de color creado a la medida de nuestras impresoras
  • Contenedores de texto ajustables
  • Importación de formato de texto enriquecido (RTF) para libros basados en texto
  • Importación de sus archivos de libros de Blurb existentes (creados en BookSmart o Bookify)

BookWright es increíblemente intuitivo, pero creamos una guía de usuario detallada para ayudarlo a comenzar.
