Uma ferramenta de criação para o artista que existe dentro de cada um

O BookWright é um aplicativo avançado de design e layout com muitos recursos para a criação de livros fotográficos, livros comerciais, revistas e e-books. Simples de usar — e gratuito Os recursos incluem:

  • Layouts gratuitos para download feitos por profissionais (e a opção de criar seus próprios layouts)
  • Livro impresso, e-book e PDF — tudo isso a partir do mesmo arquivo de projeto.
  • ISBNs exclusivos e gratuitos para cada versão do seu livro (no valor de US$ 125).
  • Saída de impressão WYSIWYG com um perfil de cores desenvolvido especialmente para nossas impressoras.
  • Caixas de texto de fluxo contínuo
  • Importação de arquivos Rich Text Format (RTF) para livros com grande volume de texto
  • Importação de seus arquivos de livro existentes do Blurb (criados no BookSmart ou Bookify)

Achamos o BookWright incrivelmente intuitivo, mas mesmo assim criamos um guia do usuário detalhado para ajudar você a começar a usá-lo.
